# Teslo API


1. Clonar el proyecto 

2. Copiar y pegar el archivo ```env.template``` y renombrarlo a ```.env``` 

3. Cambiar los valores de las variables de entorno 

4. Intalar dependencias de node

```
npm i 
```

5. Levantar la base de datos 

```
docker-compose up -d
```

6. Iniciar la aplicacion en modo desarrollo 

```
npm run start:dev
```
